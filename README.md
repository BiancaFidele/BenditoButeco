# 🍽️ Bendito Buteco – Cardápio Digital Responsivo

> Projeto acadêmico desenvolvido para a disciplina de **Engenharia de Software** (2 semestre) do curso de **Análise e Desenvolvimento de Sistemas** da **Fatec Taquaritinga**

O projeto consiste em um **cardápio digital responsivo**, desenvolvido para dispositivos móveis e desktop, permitindo que clientes realizem a montagem de seus pedidos de maneira intuitiva.

---

# 📋 Objetivo

Desenvolver uma aplicação web capaz de permitir ao usuário:

* Visualizar o cardápio do restaurante;
* Escolher entre marmitas e pratos;
* Personalizar os pedidos;
* Selecionar bebidas;
* Navegar entre as telas de forma intuitiva;
* Utilizar a aplicação tanto em computadores quanto em dispositivos móveis.

> Nesta primeira versão o projeto não possui integração com banco de dados, sendo voltado para o desenvolvimento da interface e experiência do usuário.

---

# 🚀 Tecnologias utilizadas

* HTML5
* CSS3
* Bootstrap
* JavaScript

---

# 👩🏽 Minha participação

Atuei principalmente na etapa de levantamento de requisitos, modelagem e desenvolvimento da interface.

### Principais contribuições

* Desenvolvimento da tela de seleção das marmitas, apresentando tamanhos, descrição e valores.
* Desenvolvimento das telas de personalização das marmitas (pequena, média e grande), permitindo seleção de misturas, acompanhamentos e inclusão de observações.
* Elaboração dos Diagramas de Caso de Uso (UML), utilizados na modelagem das funcionalidades.
* Participação na implementação da interface utilizando HTML, CSS, Bootstrap e JavaScript.

---

# 📱 Funcionalidades

* Visualização das opções de marmitas;
* Personalização dos pedidos;
* Seleção de misturas;
* Seleção de acompanhamentos;
* Inclusão de observações;
* Escolha de bebidas;
* Controle de quantidade de bebidas;
* Navegação entre as telas;
* Interface responsiva.
