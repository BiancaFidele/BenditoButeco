🍽️ Bendito Buteco – Cardápio Digital

Projeto desenvolvido para a disciplina de Engenharia de Software (2º semestre da graduação em Análise e Desenvolvimento de Sistemas – Fatec Taquaritinga).

O objetivo foi desenvolver um cardápio digital responsivo para um restaurante, permitindo que os clientes montassem seus pedidos de forma simples tanto pelo computador quanto pelo celular.



📋 Objetivo

Desenvolver uma aplicação web que possibilitasse:

Visualização do cardápio
Escolha entre marmitas e pratos
Seleção de bebidas
Montagem do pedido
Interface intuitiva para dispositivos móveis e desktop

Nesta primeira versão o projeto não utiliza banco de dados, sendo focado na interface e experiência do usuário.



🚀 Tecnologias utilizadas
HTML5
CSS3
Bootstrap



## 👩‍💻 Minha participação

Fiquei responsável por atividades relacionadas ao levantamento de requisitos, modelagem e desenvolvimento da interface.

Principais contribuições:

- Desenvolvimento da tela de seleção das marmitas, apresentando os tamanhos disponíveis, descrição dos produtos e respectivos valores.
- Desenvolvimento das telas de personalização das marmitas pequena, média e grande, permitindo a seleção de misturas, acompanhamentos e inclusão de observações.
- Elaboração dos Diagramas de Caso de Uso (UML), utilizados na modelagem das funcionalidades do sistema.
- Participação na implementação da interface utilizando HTML, CSS, Bootstrap e JavaScript, seguindo o padrão visual definido pela equipe.



## 📱 Funcionalidades

* Visualização das opções de marmitas com descrição e preços.
* Personalização dos pedidos conforme o tamanho da marmita.
* Seleção de misturas e acompanhamentos.
* Inclusão de observações no pedido.
* Escolha de bebidas e controle de quantidade.
* Navegação entre as telas do sistema.
* Interface responsiva para dispositivos móveis e desktop.

## 📚 Aprendizados

Durante o desenvolvimento deste projeto tive a oportunidade de aplicar conceitos de Engenharia de Software, trabalhando desde a modelagem do sistema até a implementação das interfaces. O projeto contribuiu para aprimorar conhecimentos em HTML, CSS, Bootstrap, organização de requisitos, diagramas UML e desenvolvimento colaborativo em equipe.
